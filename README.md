# calculadora-impuestos-android
Aplicación móvil nativa para Android desarrollada en MIT App Inventor.
Proyecto de desarrollo móvil nativo enfocado en la automatización de cálculos financieros basados en las reglas fiscales aplicadas a los ingresos del usuario.

   ## 🛠️ Entorno de Desarrollo
   - **Plataforma:** MIT App Inventor (Bloques lógicos y arquitectura Android)
   - **Formato del archivo:** `.aia` (Código fuente del proyecto)

   ## 🧠 Conceptos de Ingeniería Aplicados
   - **Estructuras de Control:** Implementación de lógica condicional compleja para la segmentación de tasas impositivas.
   - **Manejo de Datos:** Uso de variables globales y locales para el procesamiento de fórmulas financieras.
   - **Diseño de Interfaz (UX/UI):** Creación de una interfaz responsiva y dinámica que valida los datos de entrada del usuario en tiempo real.
